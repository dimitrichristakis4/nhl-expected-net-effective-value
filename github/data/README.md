# Data Setup

Raw data files go in this folder. None are included in the repo.

## shots_2022.csv (MoneyPuck)

Free download at moneypuck.com. Go to the shots data section and grab the 2022-23 regular season file. Save it here as `shots_2022.csv`.

## 2022_2023_NHL_PbP_Logs.xlsx / .csv (BigDataBall)

Paid dataset from bigdataball.com. Script 01 reads the `.xlsx` version. Script 05 reads a `.csv` version of the same file. Check the path constants at the top of each script.

## games-2.csv

Team-level 2022-23 stats with at minimum: Team, GF, xGF, CF%. Used only by the validation script. Download from Natural Stat Trick or equivalent.
